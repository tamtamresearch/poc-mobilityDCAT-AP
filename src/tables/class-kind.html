Definition A description following the VCARD-RDF specification, e.g., to provide telephone number and e-mail address for a contact point.

Usage note

This class is the parent class for the four explicit types of vCards (vcard:Individual, vcard:Organization, vcard:Location, vcard:Group).

The contact point might be populated with the same information as for an agent, see class foaf:Agent. See for further usage notes on that matter.

When providing contact information, data publishers SHOULD prioritize organizational information over personal details. The exposure of personal information, such as a personal name and email address, may be restricted to the public. Instead, there might be an anonymous contact channel, e.g. a contact button on the GUI that triggers an email to the agent. See .

Properties

For this entity the following properties are defined: address, affiliation, email, name, phone, URL .
